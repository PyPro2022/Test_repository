Описание приложения Bitcoin rate:

Это приложение - упрощённая версия версия учебного проекта Flask mini library,
которое мы разрабатывали на лекциях.

Модули:
/
    utils.py - содержит импорты и методы для работы с данными
    app.py - содержит импорты и вью функции для передачи и отображения данных в соттв. шаблонах


Шаблоны:
/
    templates/
         rates/
             s_search_res.html - шаблон страницы результатов поиска.*
             info.html - шаблон справочной страницы
        index.html - шаблон основной страницы
        s_base.html - базовый шаблон. *

       ---------------------------------
        * -буква s вначале для того, чтобы шаблон не смешивался с другим шаблонами с аналогичным
                        именем из других проектов, которые просматривал в процессе разработки
